    registerHeaderComponent

    registerHeaderComponent: (component: ReactNode) => void

    Register a component to be displayed in the header

    Components can call this from useEffect to add themselves to the header. Returns the component to the list when the effect runs.

    Type Declaration

      • (component: ReactNode): void

      • Parameters

        • component: ReactNode

          React component or element to register

        Returns void

    Example

    const MyComponent = () => {
      const { registerHeaderComponent, deregisterHeaderComponent } = useHeader();

      useEffect(() => {
        const button = <Button>Action</Button>;
        registerHeaderComponent(button);
        return () => deregisterHeaderComponent(button);
      }, []);

      return <div>Content</div>;
    };

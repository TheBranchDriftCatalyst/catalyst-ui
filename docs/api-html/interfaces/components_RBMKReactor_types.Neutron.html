    Interface Neutron

    Free neutron particle

    Real physics:

    • Thermal neutrons: ~2200 m/s velocity
    • Fast neutrons: ~20,000 km/s velocity (pre-moderation)
    • Average lifetime in reactor: ~0.1 seconds
    • U-235 fission releases 2.43 neutrons on average

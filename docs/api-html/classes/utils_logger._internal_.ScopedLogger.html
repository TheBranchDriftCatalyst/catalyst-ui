    Class ScopedLogger

    ScopedLogger - Logger instance bound to a specific source/component

    Provides simplified logging API without requiring source parameter on every call. Automatically registers with LoggerRegistry for runtime control via LoggerControl UI.

    See

    • createLogger - Primary way to create ScopedLogger instances
    • LoggerRegistry - Global registry for runtime logger control

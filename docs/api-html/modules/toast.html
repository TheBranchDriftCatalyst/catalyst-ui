    Module toast

    Toast UI Components

    React components for displaying toast notifications built on Radix UI Toast primitives. Includes visual variants, animation styles, and an integrated progress indicator.

    Example

    import { Toaster } from "@/catalyst-ui/ui/toaster";

    function App() {
      return (
        <>
          {/* Your app content */}
          <Toaster />
        </>
      );
    }
